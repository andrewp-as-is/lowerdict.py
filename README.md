### Installation
```bash
$ [sudo] pip install lowerdict
```

#### Examples
```python
from lowerdict import lowerdict

# lowerdict(*args, **kwargs)
lowerdict({"KEY": "value", "KEY2": "value2"})
>>> {"key": "value", "key2": "value2"}
```
